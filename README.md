# Trending GoRails

[GoRails](https://gorails.com) episodes in one page with [Mithril](https://lhorie.github.io/mithril/).

- List all episodes.
- Order by hearts, sequential_id or title.
- Filter with pro episode or not.

## Development

### Fetch

```
$ rake fetch
```

### Server

```
$ ruby -run -e httpd . -p 8000
```
